# Maestro-Style Website Design Specification

## Luxury Corporate Editorial UI/UX Guide

---

## 1. Overview

This document defines the design system, layout structure, and UI/UX principles required to build a website similar to the Maestro Joinery style.

The target style is:

- Luxury
- Minimalist
- Editorial
- Architecture / Interior-focused
- Brand-first (not conversion-heavy)

Primary goal:

> Communicate craftsmanship, precision, and premium quality through visual storytelling.

---

## 2. Design Principles

### 2.1 Core Philosophy

- Less UI, more content
- Large imagery over dense text
- Space = luxury
- Typography = brand identity
- Motion = subtle and controlled

### 2.2 Visual Keywords

- Refined
- Timeless
- Crafted
- Precision
- Bespoke
- Elegant
- Architectural

---

## 3. Color System

### 3.1 Base Palette

```txt
Background: #F7F5F0 / #FAF8F4
Primary Text: #111111
Secondary Text: #5F5A54
Border: #D8D1C7
```

### 3.2 Accent (Optional)

```txt
Gold muted: #B39A72
Warm brown: #8D7557
```

### 3.3 Rules

- Avoid bright colors
- No gradient-heavy UI
- Use neutral tones only
- Maintain high contrast readability

---

## 4. Typography System

### 4.1 Font Pairing

Option A (Recommended – Luxury):

- Heading: Serif (Playfair Display / Cormorant / Canela)
- Body: Sans-serif (Inter / Manrope / Helvetica)

Option B (Modern Minimal):

- Heading + Body: Inter / Sora / General Sans

### 4.2 Scale

```txt
Hero Title: 56–84px
Section Title: 32–48px
Body Text: 16–18px
Meta Text: 12–14px (UPPERCASE)
```

### 4.3 Typography Rules

- Use uppercase for labels/meta
- Letter spacing: 0.12em–0.2em for small text
- Line height: 1.5–1.8
- Max text width: 720–820px

---

## 5. Layout System

### 5.1 Grid

- 12-column grid
- Max width: 1280–1440px
- Container padding: 24–40px

### 5.2 Spacing

```txt
Section spacing: 96–160px
Component spacing: 24–40px
```

### 5.3 Layout Rules

- Never full-width text blocks
- Always use breathing space
- Avoid clutter
- Prioritize visual hierarchy

---

## 6. Core Components

### 6.1 Header

- Left: Logo
- Right: Navigation
- Transparent or white background
- Sticky on scroll

Menu:

- About
- Services
- Projects
- Clients
- News
- Contact

### 6.2 Hero Section

- Fullscreen image (70–100vh)
- Minimal overlay
- Bottom-left text alignment

Content:

- Category (optional)
- Title (large)
- Subtitle (1–2 lines)

### 6.3 Content Sections

Pattern:

- Text + Image (2-column)
- Alternating layout
- Large spacing

### 6.4 Project Card

- Large image
- Title
- Location / Year
- No heavy shadow
- Hover: slight zoom

### 6.5 News Card

- Thumbnail
- Date (small uppercase)
- Title (2–3 lines)
- Short excerpt

### 6.6 Logo Wall

- Grayscale logos
- Grid layout
- Equal spacing

### 6.7 Footer

- Dark or warm neutral background
- 3–4 columns:
  - Navigation
  - Contact
  - Social
  - Legal

---

## 7. Image Style Guidelines

Images must be:

- High resolution
- Architectural/interior focused
- Natural lighting
- Neutral tones
- Minimal clutter

Avoid:

- Stock business photos
- Overexposed images
- Bright saturated colors

---

## 8. Motion & Interaction

### Allowed

- Fade-in on scroll
- Subtle hover zoom (1.03–1.08)
- Smooth transitions
- Light parallax

### Not Allowed

- Bounce animations
- Flashy effects
- Heavy motion
- Neon/glow effects

---

## 9. UI Elements

### Buttons

Primary:

- Black background
- White text

Secondary:

- Outline style

Rules:

- Uppercase text
- Wide padding
- Minimal border radius

### Borders

- 1px only
- Neutral color
- Prefer border over shadow

### Radius

```txt
0px – 12px max
```

## 13. Common Mistakes to Avoid

- Too many colors
- Too many icons
- Heavy shadows
- Bright CTA buttons
- Dense text blocks
- Over-animation
- Cheap fonts
- Low-quality images
- Small spacing

---

## 14. Summary

This design style is:

> Minimal, spacious, image-driven, typography-focused, and emotionally premium.

Priority order:

1. Image quality
2. Spacing
3. Typography
4. Layout
5. Motion

---

## 15. Quick Build Brief for Agent

Build a website in a luxury corporate editorial style inspired by premium interior, architecture, and craftsmanship brands. Use large imagery, generous whitespace, refined typography, muted neutral tones, minimal motion, and a brand-first layout. Avoid conversion-heavy blocks, flashy animations, bright colors, and dense UI. Prioritize visual storytelling, project showcases, trust signals, and elegant content presentation.
