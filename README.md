# Restaurant Fit-out Platform

Monorepo scaffold for the Industrial Minimal restaurant construction website.

## Apps

- `apps/web`: Next.js frontend with locale routing and starter UI
- `apps/cms`: Strapi skeleton for content modeling and admin setup

## Getting started

```bash
pnpm install
pnpm dev:web
```

To start the CMS later:

```bash
pnpm dev:cms
```

## Current scope

- Workspace and app structure
- Frontend locale routing for `vi` and `en`
- Base design tokens and starter marketing pages
- CMS skeleton with placeholder config files
